Project: SSM Party Planner

Requirements:
1. Python 3.10+
2. MySQL

Setup Steps:

1. Import ssm_party.sql into MySQL

2. Install required packages:
   pip install flask mysql-connector-python

3. Open app.py and update MySQL settings if needed:
   host="localhost"
   user="root"
   password="your_password"
   database="ssm_party"
   port=3308

4. Run the project:
   python app.py
